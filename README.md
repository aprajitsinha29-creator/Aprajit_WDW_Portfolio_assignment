# 🌐 Personal Portfolio — Aprajit Shankar Sinha

This is my personal portfolio website built using **pure HTML and external CSS only**.  
There is **no JavaScript**, keeping the website simple, fast, and fully assignment-friendly.

It includes all required sections such as Home, About, Skills, Projects, and Contact.

---

## 🚀 Features

- Clean and simple UI  
- Fully responsive design  
- External CSS only (no internal CSS, no JS)  
- Modern layout using HTML5 semantic tags  
- Skill bars built with pure CSS  
- Project cards with clean UI  
- Contact form (non-functional, no backend required)  

---

## 🌟 How to Use

1. Download or clone the repository  
2. Make sure `index.html` and `style.css` are in the same folder  
3. Put your profile image as **profile.jpg**  
4. Open `index.html` in any browser  

That's it — your portfolio is live locally!
